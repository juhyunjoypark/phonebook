프로그램의 목적: Java Language 를 통하여 전화번호부 시스템을 구현하였다. 전화번호부에 저장되는 사람들의 속성을 일반, 대학, 회사 별로 구별하고 각자의 사람들에 대하여 필요한 요소를 추출하여 클래스 필드 값으로 할당하였다. Serialization 을 사용하여 인스턴스를 Store & Load 하도록 구현하여 프로그램이 종료된 이후에도 정보를 지속적으로 사용할 수 있도록 관리하였으며, User Interface 관점을 고려할 수 있도록 GUI 를 이용하여 현재 저장된 사람들을 검색 & 삭제 할 수 있도록 Interface 를 구현하였다. 각 사람들은 콜렉션 객체의 HashSet 을 이용하여 동적으로 관리되어지고 있으며, 공통적인 객체는 추출하여 캡슐화 , 상속관계를 형성하도록 하였으며, 같은 Action 에 대해 다르게 동작할 수 있도록 Overriding 을 사용하여 동작하도록 설계하였다.
